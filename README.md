# Project README

> **Note:** The `.env` and `.secrets` configuration details are included in the Application Workflow document/file linked below

## Overview

This project consists of a frontend and backend application.

* Frontend: React/Vite application
* Backend: Django application
* 

The application can be run locally by starting both the frontend and backend servers.

---

# Setup Instructions

## 1. Clone the Repository

```bash
git clone https://github.com/KatyayniUpadhyay/personalized-greeting-application
```

---

# Frontend Setup

Open a terminal and run:

```bash
cd frontend
npm install
npm run dev
```

The frontend server will start locally.

---

# Backend Setup

Open another terminal and run:

```bash
cd backend
python manage.py migrate
python manage.py runserver
```

The backend server will start locally.
